# Framells Photo Booth

Framells is a simple web-based photo booth application that allows users to capture photos using their device camera directly in the browser. The project demonstrates how modern web technologies can interact with hardware like cameras to create fun and interactive experiences.

---

## Live Demo

https://yash-dev05.github.io/framells-photo-booth/

---

## Features

• Capture photos using the device camera
• Instant preview of captured images
• Clean and simple user interface
• Works directly in the browser
• No installation required

---

## Technologies Used

HTML5
CSS3
JavaScript
Web Camera API

---

## Project Structure

index.html – Main structure of the application
style.css – Styling and layout of the interface
script.js – Camera access and application logic
assets/ – Images or frame assets used in the project

---

## How It Works

1. The website asks permission to access the user's camera.
2. Once permission is granted, the camera feed appears on the screen.
3. The user captures a photo using the capture button.
4. The captured image is displayed instantly in the interface.
